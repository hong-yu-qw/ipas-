# 集線器 (HUB)
~~~
是一個提供集中網路內多條纜線.並傳輸所有資料到所有裝置中央裝置
~~~
# 交換器 (switch)
~~~
類似集線器.資料只傳輸給預先決定的收件方
~~~
# 路由器 router
~~~
是一種將二或更多個網路相連接.並負責
~~~
# 網路介面卡 (network interface card) NIC
~~~
是一張能將電腦連上有線或無線網路的電路板
~~~
# DSL數據機 (modem)
~~~
是一個將數位訊號調變到類比訊號上進行傳輸，並解調收到的類比訊號以得到數位訊號的電子裝置
~~~

# 虛擬私人網路 (virtual private network)
~~~
是一種常用於連接中、大型企業或團體與團體間的私人網路的通訊方法
它利用隧道協定（Tunneling Protocol）來達到傳送端認證、訊息保密與準確性等功能。
